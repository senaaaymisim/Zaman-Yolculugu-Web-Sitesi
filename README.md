# 🚀 Zaman Yolculuğu Web Sitesi

Bu proje, 1970–2010 yılları arasındaki unutulmaya yüz tutmuş müzikleri ve nostaljik teknolojileri tanıtmak amacıyla hazırlanmış bir PHP destekli web sitesidir.

## 📌 Proje Hakkında

Zaman Yolculuğu, kullanıcıları geçmiş yıllara götüren interaktif bir nostalji platformudur.

Sitede:

- 1970, 1980, 1990, 2000 ve 2010 yıllarının popüler müzik türleri gösterilir.
- Yıllara tıklandığında ilgili YouTube içeriklerine yönlendirme yapılır.
- Walkman, Atari, MSN Messenger, iPod, Plak Çalar gibi nostaljik teknolojiler tanıtılır.
- Kullanıcılar anılarını paylaşabilir.
- Paylaşılan anılar ayrı bir sayfada görüntülenebilir.
- Nostalji Testi ile kullanıcılar bilgilerini ölçebilir.
- Ziyaretçi sayacı bulunur.
- PHP kullanılarak dinamik içerikler oluşturulmuştur.
